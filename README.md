File updates included in sequential order of modification.

This code was written to digitally reconstruct lunar craters using images of a single laser over a controlled
drop chamber. Initial versions used a prototype while more recent tests have started analysing drops and crater formation.

Analysis 1 and 2 got the basics down and were filled with noise, images were taken on a prototype.

Analysis 3-5 werre within the vacuum chamber and provided significantly more focused results.

Analysis 6 was of a pre-determined shape within the chamber and introduces a digital scale calibration.

Future versions intend to introudce analytical capabilities and calculate impact data.
